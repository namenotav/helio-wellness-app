   - Search exercise names
   - Free for commercial use
   - No attribution needed

3. **Giphy Fitness** - https://giphy.com/
   - Search "workout" + exercise name
   - Check license (some require attribution)

4. **Fitness GIF APIs:**
   - ExerciseDB (https://exercisedb.io/) - API with exercise GIFs
   - Wger (https://wger.de/) - Free workout database with images
